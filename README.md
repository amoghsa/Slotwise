# Slotwise

Public home for **Slotwise** — support, privacy, and data-deletion information.

Slotwise helps a parent build a **conflict-free weekly activity schedule** for
their kids each season, then export it to the calendar they already use. It works
fully offline, has no accounts, and **keeps all your data on your device**.
